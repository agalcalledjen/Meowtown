## Setup

### Server

**NOTE: Please note that a database will be required to run the server.**

Commands must be run from the `server` directory:

#### Installation

```bash
npm install
```

#### Run

```bash
npm run start:dev
```

---

## Client

Commands must be run from the `client` directory:

#### Installation

```bash
npm install
```

#### Run

```bash
npm start
```

#### Build

```bash
npm run build
```
